This default Utility panel has the title "My Utility", and the text "Utility Command" at the top, followed by three text input boxes. From top to bottom these are labeled "Executable", "Options", and "CGNS Input". The Executable" and "CGNS Input" boxes have "Browse" buttons to their right. In this example the first and third boxes contain the text "print_cgns" and "CGNSinput". The "Options" box is empty. Below the last input box is a check box labeled "Run Command in Background".

At the bottom are "Accept" and "Cancel" buttons.
