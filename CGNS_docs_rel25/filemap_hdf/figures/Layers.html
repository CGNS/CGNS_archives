This diagram is intended to illustrate the relationship between the specification and software aspects of CGNS. On the left are four boxes, with the following labels:

  • SIDS
  • MLL tree interface
  • SLL node interface
  • SLL mapping to HDF5
On the right there are also four boxes, with the following labels:
  • End User Application
  • MLL (Trees)
  • SLL (Nodes)
  • Storage layer (HDF5 or ADF)

The "SIDS" box on the left points to the "End User Application" box on the right. The "MLL tree interface" box on the left points to the "MLL (Trees)" box on the right. The "SLL tree interface" and "SLL mapping to HDF5" boxes on the left both point to both the "SLL (Nodes)" box on the right.
