This chart shows a typical CGNS file in tree form. At the top is the Root node, with child nodes CGNSLibraryVersion_t and CGNSBase_t. Below CGNSBase_t are the following child nodes.

  • Zone_t, with the node name Zone 1
  • Zone_t, with the node name Zone 2
  • ConvergenceHistory_t
  • ReferenceState_t
Below one of the Zone_t nodes are:
  • Elements_t
  • GridCoordinates_t
  • ZoneBC_t
  • FlowSolution_t
  • ZoneGridConnectivity_t
Below the Elements_t node is a DataArray_t node named ElementConnectivity. Below the GridCoordinates_t node are two DataArray_t nodes named CoordinateX and CoordinateY. Below the FlowSolution_t node is a GridLocation_t node named GridLocation, and two DataArray_t nodes named Density and Pressure.
