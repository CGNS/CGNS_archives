This is a plot of the creation wall clock time in seconds vs the number of processors. There are three curves, one for the small-grid case (50×50×50 in each zone), one for the medium-grid case (150×150×150 in each zone), and one for the large-grid case (250×250×250 in each zone). For all three grids, the time increases linearly with the number of processors, as follows:

  • For the small-grid case, from nearly 0 for 1 processor to about 0.5 for 16 processors
  • For the medium-grid case, from about 0.2 for 1 processor to about 3.3 for 16 processors
  • For the large-grid case, from about 0.7 for 1 processor to about 11.7 for 16 processors
