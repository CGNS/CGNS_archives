This figure illustrates the high-level structure of a CGNS database.

At the top is the base node of the CGNS database, labeled CGNSBase_t. Below CGNSBase_t are the following nodes.

  • Zone_t
  • ZoneUnstructured_t
  • FlowEquationSet_t
  • Family_t
  • Descriptor_t
  • ReferenceState_t
  • DataClass_t
  • DimensionalUnits_t
  • IndexDimension
  • ConvergenceHistory_t

Below the Zone_t node are the following nodes.

  • NCells, Nvertices
  • GridCoordinates_t
  • FlowSolutions_t
  • ZoneGridConnectivity_t
  • ZoneBC_t
Additional nodes may also be below Zone_t.

Below the ZoneUnstruct_t node are the following nodes.

  • NCells, Nvertices
  • GridCoordinates_t
  • FlowSolutions_t
  • ZoneGridConnectivity_t
  • ZoneBC_t
  • Elements_t
Additional nodes may also be below ZoneUnstruct_t.

Below the FlowEquationSet_t node are the following nodes.

  • GoverningEquations_t
  • ViscosityModel_t
  • TurbulenceModel_t
  • GasModel_t
  • ThermalConductivityModel_t
  • TurbulenceClosure_t
  • Descriptor_t
  • EquationDimension

Below the Family_t node are the following nodes.

  • FamilyBC_t
  • GeometryReference_t
  • Descriptor_t
