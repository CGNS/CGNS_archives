This figure shows the Family_t node structure, with the node name FamilyName, below the CGNSBase_t node. There are two child nodes of Family_t: GeometryReference_t and FamilyBC_t. In turn, the GeometryReference_t has three children: GeometryFile_t, GeometryFormat_t, and GeometryEntity_t. The FamilyBC_t has one child, BCType_t.
