This figure is an image of the ADFviewer main window. The window is divided into several areas, or panels, each described separately.

Along the top are the Menu Items, followed by a series of Toolbar Buttons. Below these, on the left is the expandable/collapsible Node Tree. On the right, from top to bottom, are the Node Description, Link Description, and Data Description panels, a row of node-related Buttons, and the Node Data panel. A Separator Bar is located between the Node Tree on the left, and the various panels on the right.
