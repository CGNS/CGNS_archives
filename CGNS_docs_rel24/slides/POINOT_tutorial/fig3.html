This diagram shows a CGNS node tree illustrating the storage of data for an example of code coupling.

Under the ZoneBC_t node for a zone, there's a BC_t node named "Wall:Heat". Below Wall:Heat is a PointRange node, and a BCDataSet_t node named ".Solver#BC". Below the .Solver#BC node are nodes named "type", "wall_temp", "Protocol", and "Path".

Under the FlowSolution_t node for the zone, there's a UserDefinedData_t node named ".Solver#SolutionSubSetlnBC". Below that node is a node named "Wall:Heat", with child nodes named "Protocol", "PathList", and "PointRange".
