This is an image of the ADFviewer main window. The window is divided into several areas, or panels.

Along the top are the menu items, followed by a series of toolbar buttons. Below these, on the left is the expandable/collapsible node tree. On the right, from top to bottom, are the node description, link description, and data description panels, a row of node-related buttons, and the node data panel.

At the top of the node tree is the root node of the ADF file, named simply "/", and shown with a folder icon. The next line is the "CGNSLibraryVersion" node, marked with a file icon.

Next is the base node for the CGNS database, named "Base" and marked with a folder icon. The left is a "-" sign, indicating that the Basename node has been expanded, and clicking the "-" sign will collapse it.

Below the Base node are the following nodes, with the names indicated.

  • "Zone 1", marked with a folder icon, and collapsed
  • "InputFileUsed 1", marked with a file icon
  • "SimulationType", marked with a file icon
  • "GlobalConvergenceHistory", marked with a folder icon, and collapsed
  • "DataClass", marked with a file icon
  • "ReferenceState", marked with a folder icon, and collapsed
  • "TimeIterValues", marked with a folder icon, and collapsed
  • "CFL3DTimeStep", marked with a folder icon, and collapsed
  • "InputFileUsed 2", marked with a file icon
  • "Case Title", marked with a file icon

In the node tree the Zone 1 node is highlighted. In the node description panel, the parent node value shown is "/Base", the node name is Zone 1, and the node label is Zone_t. In the data description panel, the data type value is I4, the dimensions values are 3 and 3, and the bytes value is 16. Node values are shown in the node data panel.
