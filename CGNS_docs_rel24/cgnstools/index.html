CFD General Notation System
CGNS Tools and Utilities

Document Version 1.0.2

A variety of CGNS-related utility packages have been developed, both by CGNS team members and by CGNS users.

ADFviewer
An ADF/CGNS file viewer and editor; requires Tcl/Tk 8.3. The GUI allows access to any node in the file using a Windows-like collapsible node tree. Nodes and data may be added, deleted, and modified. Several other utilities may be accessed from ADFviewer, including:

  • CGNSplot, for displaying the surface mesh, element sets, connectivity regions, and boundary condition regions
  • CGNScalc, a calculator using data in CGNS files
  • CGNScheck, a CGNS file validator
  • Import, export, data conversion, and subset extraction and interpolation utilities

ADFviewer is part of the CGNStools package, available from SourceForge .
Contact: Bruce Wedan (brucewedan@hotmail.com)

CGNSplot
For structured zones, CGNSplot may be used to display the exterior faces of the volume mesh, the minimum and maximum bounding surfaces, the connectivity regions, and the boundary condition regions. For unstructured zones, CGNSplot will display the element sets. It may be run stand-alone, or accessed from ADFviewer.

CGNSplot is part of the CGNStools package, available from SourceForge .
Contact: Bruce Wedan (brucewedan@hotmail.com)

Utilities
A collection of import, export, data conversion, and subset extraction and interpolation utilities that may be run stand-alone, or accessed from ADFviewer.

  • Utilities for converting Patran, PLOT3D, or Tecplot files to CGNS files, and CGNS files to PLOT3D or Tecplot files
  • Data conversion utilities for modifying the grid location (vertex or cell-center), solution variables (primitive or conservative), and data class (Dimensional, NormalizedByDimensional, or NormalizedByUnknownDimensional)
  • Utilities for creating a subset or interpolated version of a CGNS file

These utilities are included in the CGNStools package, available from SourceForge .
Contact: Bruce Wedan (brucewedan@hotmail.com)

ADF_Edit
A browser/editor that displays the ADF tree structure and contents in ASCII form, and permits item-by-item editing. Although ADF_Edit is still available from SourceForge, it has been superseded by the much more capable ADFviewer.

CGNS_readhist
Reads a CGNS file and writes history data to a formatted file. CGNS_readhist is available from the Utilities page at the CGNS web site .
Contact: Chris Rumsey (c.l.rumsey@larc.nasa.gov)

FTU (File Transfer Utility)
Converts CGNS files to and from Plot3D format. Has a text-based menu allowing the manipulation of a CGNS base. FTU is available from the Utilities page at the CGNS web site .
Contact: Bob Bush (bushrh@pweh.com)

CGNS Viewer
An alpha release of another ADF/CGNS file viewer and editor. Requires the GTK+ library version 1.2 or newer (installed by default on newer Linux systems using Gnome; otherwise found at http://www.gtk.org/ ). CGNS Viewer is available from the Utilities page at the CGNS web site .
Contact: Christian Lundh (xistan@yahoo.se)

PyCGNS
A Python binding to the ADF Core and Mid-Level Library.
Contact: Marc Poinot (Marc.Poinot@onera.fr)

ADFM
A development version of software for managing an in-memory representation of ADF trees. For more information, see "Proposal and Prototype for an in-memory representation of ADF trees " (PDF, 7 pages, 256K).
Contact: Marc Poinot (Marc.Poinot@onera.fr)

CGNS++
A C++ binding to the ADF Core and Mid-Level Library.
Contact: Manuel Kessler (kessler@iag.uni-stuttgart.de)

The complete CGNS Tools and Utilities documentation is available for installation on a local system. This is a gzip'ed tar file (318K, 635K gunzip'ed), and may be downloaded by clicking on the link while holding down the shift key, assuming your browser supports this capability. A dialogue box will appear for you to specify the directory and file name. To unpack the contents, do 

   gunzip -c cgnstools.tar.gz | tar xvf -
The files will be extracted and stored in the directory cgnstools. Point your browser to the local file cgnstools/index.html to display the CGNS Tools and Utilities home page. Note that some links, to other manuals, etc., may not work.
