Base Selection

The base selection entry allows the desired CGNS base to be selected for display. Clicking Mouse Button-1 in the entry field or on the pull-down arrow, will invoke a selection list displaying all the bases in the CGNS file. When a base is selected, the graphics will be reset, and the Zone List will be filled with the list of zones under the base.

Clicking Mouse Button-3 in the base selection field will pop-up a window similar to the one below, listing the properties of the current CGNS base.
