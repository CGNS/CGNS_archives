This default Import panel has the title "My Import", and the text "Import Command" at the top, followed by four text input boxes. From top to bottom these are labeled "Executable", "Options", "Input File", and "CGNS Output". All but "Input File" have "Browse" buttons to their right. In this example these four boxes contain the text "import_my_file", "option1 option2", "InputFile", and "CGNSOutput".

At the bottom are "Accept" and "Cancel" buttons.
