This default Command panel has the title "My Command", and the text "Run Command" at the top, followed by two text input boxes labeled "Executable" and "Arguments". The Executable" box has a "Browse" button to its right. In this example the boxes contain the text "run_command" and "options". Below the last input box is a check box labeled "Run Command in Background".

At the bottom are "Accept" and "Cancel" buttons.
