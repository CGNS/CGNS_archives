This is an image of the ADFviewer main window showing the CGNS file structure after the third part of the example. It shows the same node tree as described after the second part of the example but with the GridCoordinates node expanded.

Below the GridCoordinates node are nodes named "CoordinateX", "CoordinateY", and "CoordinateZ", each marked with a file icon.
