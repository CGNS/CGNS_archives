This figure illustrates the structure of the new Family_t node, with the node name FamilyName, below CGNSBase_t.

There are two child nodes of Family_t: GeometryReference_t and FamilyBC_t. In turn, the GeometryReference_t node has four children: GeometryFile_t, GeometryFormat_t, GeometryEntity_t, and Descriptor_t. The FamilyBC_t node has one child, BCType_t.
