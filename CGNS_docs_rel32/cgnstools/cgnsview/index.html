CGNSview

The CGNSview program provides a graphical user interface to view and edit ADF and/or HDF5 based CGNS files. It reads the file using only the core cgio routines, thus should be able to handle invalid or damaged CGNS file. The current documentation is brief in nature, but hopefully adequate for most needs.
