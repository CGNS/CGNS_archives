Notes
  • Queries in lower case letters will match any case; otherwise, the search is case sensitive. I.e., "elements" will match elements and Elements, while "Elements" will only match Elements.

  • Use quotation marks to search for phrases (e.g., "boundary conditions").

  • See the search help pages for additional details on the syntax used to search for multi-word phrases, perform special searches, etc. (Note that the "Search" link on those pages will take you to the search page for the entire NASA Glenn web server. Use your browser's "Back" button to return to this page to search just the CGNS documentation.)

  • Only the documentation for the current CGNS version is searched - not previous versions, and not the beta version.
