ADF User's Guide

Document Version 1.1.1

This document defines the structure of an ADF file, and describes in detail the ADF Core, a library of low-level routines used to store and retrieve data in an ADF file.

  • Database-Level Routines
    • ADF_Database_Open - Open a database
    • ADF_Database_Close - Close an opened database
    • ADF_Database_Delete - Delete an existing database
    • ADF_Database_Get_Format - Get the data format in an existing database
    • ADF_Database_Set_Format - Set the data format in an existing database
  • Data Structure and Management Routines
    • ADF_Create - Create a node
    • ADF_Delete - Delete a node
    • ADF_Children_Names - Get the child names of a node
    • ADF_Number_of_Children - Get the number of children of a node
    • ADF_Get_Node_ID - Get a unique identifier of a node
    • ADF_Get_Name - Get the name of a node
    • ADF_Put_Name - Put (change) the name of a node
    • ADF_Move_Child - Change a parent (move a child)
    • ADF_Link - Create a link
    • ADF_Is_Link - Test if a node is a link
    • ADF_Get_Link_Path - Get the path information from a link
    • ADF_Get_Root_ID - Get the root ID of the ADF file
  • Data Query Routines
    • ADF_Get_Label - Get a label
    • ADF_Set_Label - Set a label
    • ADF_Get_Data_Type - Get the data type
    • ADF_Get_Number_of_Dimensions - Get the number of dimensions
    • ADF_Get_Dimension_Values - Get the dimension values of a node
    • ADF_Put_Dimension_Information - Set the data type and dimension information
  • Data I/O Routines
    • ADF_Read_Data - Read the data from a node (with partial capabilities)
    • ADF_Read_All_Data - Read all the data into a contiguous memory space
    • ADF_Read_Block_Data - Read a contiguous block of data from a node
    • ADF_Write_Data - Write the data to a node (with partial capabilities)
    • ADF_Write_All_Data - Write all the data from a contiguous memory space
    • ADF_Write_Block_Data - Write a contiguous block of data to a node
  • Miscellaneous Routines
    • ADF_Flush_to_Disk - Flush the data to a disk
    • ADF_Database_Garbage_Collection - Garbage collection
    • ADF_Error_Message - Return an error message
    • ADF_Set_Error_State - Set the error state
    • ADF_Get_Error_State - Get the error state
    • ADF_Database_Version - Get the ADF file version ID
    • ADF_Library_Version - Get the ADF library version ID
