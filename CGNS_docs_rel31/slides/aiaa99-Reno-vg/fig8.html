This figure illustrates the structure of the new ZoneUnstructured_t node. Below the ZoneUnstructured_t node are the following nodes.

  • GridCoordinates_t
  • FlowSolutions_t
  • Elements_t
  • ZoneGridConnectivity_t
  • ZoneBC_t
Of these, the Elements_t node is new and contains unstructured grid connectivity information. The other nodes below ZoneUnstructured_t are the same as for structured grids.
