This is an image of the ADFviewer main window illustrating the storage of governing equation information in a CGNS file. It shows the same node tree as described earlier for the figure illustrating the storage of boundary condition information in a CGNS file but with the FlowEquationSet node expanded instead of the ZoneBC node.

Below the FlowEquationSet node are the following nodes.

  • "EquationDimension", marked with a file icon
  • "GoverningEquations", marked with a folder icon, and expanded
  • "GasModel", marked with a folder icon, and collapsed
  • "ViscosityModel", marked with a folder icon, and collapsed
  • "ThermalConductivityModel", marked with a folder icon, and collapsed
  • "TurbulenceClosure", marked with a folder icon, and collapsed
  • "TurbulenceModel", marked with a folder icon, and collapsed

Below the GoverningEquations node is the node DiffusionModel, marked with a file icon.

In the node tree the GoverningEquations node is highlighted. In the node description panel, the parent node value shown is "/Base/Zone 1/FlowEquationSet", the node name is GoverningEquations, and the node label is GoverningEquations_t. In the data description panel, the data type value is C1, the dimensions value is 11, and the bytes value is 11. The string NSTurbulent is shown in the node data panel.
