This is a plot of the speed-up factor vs number of processors. There are three curves, one for the 8.2 MB dataset, one for the 57 MB dataset, and one for the 417 MB dataset. All start at a value of 1.0 for 1 processor, increase to values of 1.4 to 2.0 for 4 processors, decrease to values of 0.5 to 0.8 for 8 processors, then steadily increase to values of 2.5 to 3.5 for 50 processors. The speed-up factors for the 8.2 and 57 MB datasets then decrease, to values of 2.0 and 2.5 respectively, for 60 processors. The speed-up factor for the 417 MB dataset stays about the same, at a value of 3.5, for 60 processors.
